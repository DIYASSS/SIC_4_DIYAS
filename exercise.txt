2. Apa perbedaan git reset dan git revert
Perintah git reset , akan mengembalikan file ke kondisi sebelumnya, kemudian menghapus catatan sejarah commit beikutnya. Perintah git revert mengembalikan file dengan tidak menghapus sejarah 
commit
3. Apa bedanya GIT dengan GITHUB
 Git adalah alat yang digunakan untuk membuat hidup programmer lebih mudah, sementara GitHub adalah layanan yang digunakan untuk meng-host proyek Git.
